# 📊 Research Budget Monitoring Dashboard

An interactive Power BI dashboard designed to simulate research grant budget tracking and financial monitoring across multiple projects.

Built using Microsoft Power BI, this project demonstrates how data visualisation can support research finance management, budget tracking, and expenditure analysis.

---

## Project Objective

The goal of this project is to:
- Track and compare **budget vs actual expenditure**
- Analyse spending patterns across projects and categories
- Enable interactive exploration of financial data

---

## Tools Used

- Microsoft Power BI
- Excel (data preparation)
- DAX (calculated fields)
- Data modelling and visualisation

---

## 📂 Dataset

Simulated research finance dataset including:
- Project names (A, B, C)
- Monthly time periods
- Budget allocation
- Actual spending
- Expense categories (Staff, Equipment, Consumables)

---

## 📊 Dashboard Features

### 🔹 Key Performance Indicators (KPIs)
- Total Budget
- Total Actual Spend
- Budget Variance

### 🔹 Visual Analytics
- Budget vs Actual comparison (clustered column chart)
- Monthly spending trends (line chart)
- Category-wise expenditure breakdown (pie chart)

### 🔹 Interactivity
- Slicers for filtering by:
  - Project
  - Category
  - Month

---

## 📈 Key Insights

- Enables identification of overspending at project and category level  
- Provides visibility into monthly spending trends  
- Supports financial decision-making in research environments  

---

## Skills Demonstrated

- Financial data analysis

This dashboard structure is applicable to:
